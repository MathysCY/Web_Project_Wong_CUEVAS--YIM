Academic Project

About This Project

This website is an academic project created for the Web Development module at EFREI Paris. It showcases the Computer Science Department and was built entirely with HTML5, CSS3, and vanilla JavaScript — no frameworks or external libraries.

Technology

Built With Pure Web Standards

No frameworks, no libraries. Just HTML5, CSS3, and vanilla JavaScript — as required.

🌐

HTML5

Semantic markup with proper heading hierarchy, ARIA attributes, and W3C-valid structure on every page.

🎨

CSS3

Custom design system with CSS variables, Grid, Flexbox, animations, keyframes, and responsive media queries.

⚡

Vanilla JavaScript

Pure ES6+ JS — no jQuery, no frameworks. IntersectionObserver, DOM manipulation, and event handling.

📱

Responsive Design

Mobile-first approach with viewport meta, fluid typography, and breakpoints at 640px and 1024px.

♿

Accessibility

ARIA roles and labels, keyboard navigation, focus management, skip links, and screen reader support.

🔠

Google Fonts

Inter for body text and Sora for headings — loaded externally for premium typography without a font library.

Project Details

EFREI Web Development Module — Academic Year 2025–2026

This website was created as the final project deliverable for the Web Technologies module at EFREI Paris. The objective was to design and build a multi-page showcase website for the Computer Science Department, demonstrating mastery of HTML5, CSS3, and JavaScript — including form handling, dynamic interactions, responsive design, and table manipulation.

6 HTML pages
3 CSS files
4 JS modules
0 External libraries
2 Team members

Academic Integrity: All code, content, and design work presented in this project was produced entirely by Mathys CUEVAS--YIM and Erwan WONG. No external frameworks, templates, or plagiarized content were used. All images were generated for this project. This work complies with EFREI's academic integrity policy.

Development Timeline

How We Built It

Week 1

Planning & Design

Defined site architecture (6 pages, 2-level directory), chose design aesthetic, set up CSS design token system, and created wireframes.

Week 2

CSS Foundation & JavaScript Core

Built the complete stylesheet (style.css, animations.css, responsive.css) and the four JavaScript modules including navigation, carousel, animations, and forms.

Week 3

HTML Page Development

Built all 6 HTML pages: Home, Courses, Faculty, Student Life, Contact, and About. Implemented tables, forms, and dynamic JS features on each page.

Week 4

Polish & Validation

Refined animations, fixed responsiveness issues, ran W3C validation on every page, cross-browser tested, and polished accessibility features.
