Academic Project

About This Project

This website is an academic project created for the Web Development module at EFREI Paris. It showcases the Computer Science Department and was built entirely with HTML5, CSS3, and vanilla JavaScript — no frameworks or external libraries.

Project Details

EFREI Web Development Module — Academic Year 2025–2026

This website was created as the final project deliverable for the Web Programming module at EFREI Paris. The objective was to design and build a multi-page showcase website for the Computer Science Department, demonstrating practical application of HTML5, CSS3, and JavaScript — including form handling, dynamic interactions, responsive design, and table manipulation.

6 HTML pages
3 CSS files
4 JS modules
0 External libraries
2 Team members
Development Timeline

How We Built It

Step 1

Planning & Design

Defined site architecture (6 pages, 2-level directory), chose design aesthetic, set up CSS design token system, and created wireframes.

Step 2

CSS Foundation & JavaScript Core

Built the complete stylesheet (style.css, animations.css, responsive.css) and the four JavaScript modules including navigation, carousel, animations, and forms.

Step 3

HTML Page Development

Built all 6 HTML pages: Home, Courses, Faculty, Student Life, Contact, and About. Implemented tables, forms, and dynamic JS features on each page.

Step 4

Polish & Validation

Refined animations, fixed responsiveness issues, ran W3C validation on every page, cross-browser tested, and polished accessibility features.
