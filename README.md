# EFREI Computer Science Department Website

This project is a multi-page static website for the EFREI Computer Science Department. It presents the department homepage, academic programs, faculty, student life, and contact information through a consistent visual design built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` is the home page and entry point.
- `pages/` contains the secondary pages of the site.
- `assets/css/` contains the shared stylesheets used across the site.
- `assets/js/` contains the interactive scripts for navigation, carousel behavior, animations, and forms.
- `assets/images/` contains the photos, icons, logos, and other visual assets used throughout the pages.

## Pages

- `index.html` - Home page with the main hero section, key department highlights, featured content, and navigation to the rest of the site.
- `pages/courses.html` - Presents the computer science courses and training programs, including academic tracks and specializations.
- `pages/faculty.html` - Introduces the teaching staff and research themes, with focus on faculty profiles and expertise.
- `pages/student-life.html` - Shows student clubs, campus life, events, and community activities.
- `pages/contact.html` - Provides the contact form, department contact details, location, and practical information for reaching the team.
- `pages/about.html` - Explains the project itself, the team behind it, and the purpose of the website.

## Assets

- `assets/css/style.css` contains the main design system and shared layout rules.
- `assets/css/responsive.css` handles responsive behavior across screen sizes.
- `assets/css/animations.css` adds motion and reveal effects.
- `assets/css/rebrand.css` and `assets/css/rebrand-blue.css` contain alternate visual styling variants.
- `assets/js/nav.js` controls the navigation behavior.
- `assets/js/carousel.js` powers the image carousel on the home page.
- `assets/js/animations.js` manages scroll and reveal animations.
- `assets/js/forms.js` handles form-related interactions.

## Media

The `assets/images/` folder stores the visual content used by the site, including campus images, faculty photos, hackathon images, the EFREI logo, and supporting project visuals.
